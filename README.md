# 🐒 My Userscripts (我的油猴脚本合集)

这里收集并维护了我编写的一些实用 Tampermonkey（油猴）脚本。致力于提供轻量、安全、无侵入的网页增强体验。

## 🚀 如何使用

如果你是第一次使用油猴脚本，请先安装脚本管理器：
1. 安装浏览器扩展：[Tampermonkey](https://www.tampermonkey.net/) (推荐) 或 [Violentmonkey](https://violentmonkey.github.io/)。
2. 在下方的列表中找到你需要的脚本。
3. 点击 **[安装]** 链接，脚本管理器会自动弹出安装提示，确认即可。

## 🛠️ 目录结构说明

本仓库采用独立目录管理每个脚本，方便查阅各自的源码和文档：
- `/scripts/` - 脚本主目录
  - `/脚本A/` - 包含脚本A的 `.user.js` 源码和专属 `README.md` 说明
  - `/脚本B/` - 包含脚本B的 `.user.js` 源码和专属 `README.md` 说明
